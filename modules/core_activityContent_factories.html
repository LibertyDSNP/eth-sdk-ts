Type aliases

ActivityContentAttachment

ActivityContentAttachment: ActivityContentAudio | ActivityContentImage | ActivityContentVideo | ActivityContentLink

ActivityContentAttachment represents a piece of external content associated with an ActivityContentNote, such as a picture, video, audio clip or website.

ActivityContentTag

ActivityContentTag: ActivityContentHashtag | ActivityContentMention

ActivityContentTag is either an ActivityContentHashtag or an ActivityContentMention.

Const createHash

  • createHash(content: string): ActivityContentHash

  • createHash() provides a simple factory for generating an ActivityContentHash object. This factory assumes the user intends to use a standard Keccak256 hash. To use other authentication algorithms, users should build their own ActivityContentHash objects.

    Parameters

    • content: string

      The file content to be hashed

    Returns ActivityContentHash

    An ActivityContentHash containing the keccak256 proof of the content
