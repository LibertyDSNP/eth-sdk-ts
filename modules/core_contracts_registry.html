Const isSignatureAuthorizedTo

  • isSignatureAuthorizedTo(signature: string, announcement: string | Announcement | SignedAnnouncement, dsnpUserURIOrId: string | bigint, permission: Permission, blockTag?: BlockTag, opts?: Partial<Config>): Promise<boolean>

  • isSignatureAuthorizedTo() validates an announcement, either as a string, a SignedAnnouncement or unsigned Announcement, against a signature and then checks that the signer has the permissions specified.

    throws

    MissingProviderConfigError Thrown if the provider is not configured.

    throws

    MissingRegistrationError Thrown if a registration cannot be found for the given DSNP User URI.

    throws

    MissingContractAddressError Thrown if the requested contract address cannot be found.

    throws

    InvalidAnnouncementParameterError Thrown if the announcement provided is invalid

    Parameters

    • signature: string

      the signature for the announcement

    • announcement: string | Announcement | SignedAnnouncement

      the signed, unsigned or serialized announcement

    • dsnpUserURIOrId: string | bigint

      the DSNP User URI or Id of the supposed signer

    • permission: Permission

      the permissions to check for

    • Optional blockTag: BlockTag

      (optional). A block number or string BlockTag (see https://docs.ethers.io/v5/api/providers/types/) Defaults to 0x0, which checks for "forever" permissions.

    • Optional opts: Partial<Config>

      (optional) any config overrides.

    Returns Promise<boolean>
