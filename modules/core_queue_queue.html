QueueId

QueueId: string

QueueId is an representation of an identifier used by a queuing adapter for referencing specific items in the queue which may need to be removed in the future.

Const dequeueBatch

  • dequeueBatch(dsnpType: DSNPType, count: number, opts?: Partial<Config>): Promise<DSNPBatchMessage[]>

  • dequeueBatch() takes a DSNP type and number of messages to dequeue and returns an array for inclusion in a batch file. If the number provided is zero, all messages in the queue with a matching type will be returned.

    Parameters

    • dsnpType: DSNPType

      The DSNP type of messages to dequeue

    • count: number

      The number of messages to dequeue and return

    • Optional opts: Partial<Config>

      Optional. Configuration overrides, such as from address, if any

    Returns Promise<DSNPBatchMessage[]>

    An array of DSNP messages removed from the queue
